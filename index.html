I'm an Associate Professor in the Computer Science Department at Virginia Commonwealth University. My current research interests are centered around using machine learning and natural language processing to solve problems in software engineering, specifically, to facilitate automation of a number of tasks that software developers commonly perform. For instance, my research has focused on helping developers locate the code needed to fix a specific bug and to improve how developers search for relevant documentation on the Web. I'm also interested in cybersecurity, i.e., writing secure code, and end-user programming.

At VCU, I lead the Software Improvement (SWIM) Lab.

I'm actively looking for collaborations with industry and motivated graduate and undergraduate students to join my research group. Please contact me if you are interested.


 

Research Projects

Emotions in Software Engineering Text -- Detecting emotions expressed in software engineering communication channels (e.g., comments on pull requests, issue discussions, chats) in order to reduce toxic or incivil comments, undestand developer's emotion towards a specific aspect or feature or of the project, etc. Contribute to a more sustainable open source software eco-system. [ASE 2022]
 
Changeset-based Bug Localization -- Helping developers to localize bugs (using bug reports) to the changeset that induced them. Changesets (or diffs) can be more useful for fixing bugs than static source code (e.g., methods or classes) as they encode the change that created the bug and include a (usually) meaningful message. [ICSE 2022]
 
Conversational Interfaces for Software Development -- Use conversational interfaces for 1) end-users to program cyber-physical systems (e.g., UAVs); 2) novices to find software development information on the Web; 3) developers to improve the quality of bug reporting [Elsevier IST 2022]
 
Improved Software Vulnerability Data -- Patches to known software vulnerabilities, as maintained by the National Vulnerability Dataset are often noisy or incorrect. Our work aims to understand and improve the quality of this important data source. It also aims to create a corpus of explained patches that can be leveraged in education.
